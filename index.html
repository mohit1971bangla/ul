As a learner, your first-month goal should be-

  • Learning about Web Development (eg. What is web development? Why we'll learn Web Development? How many types of web developers are in the world?)
  • Setting the goal.
  • Know about the basic things you need.
  • Make 3 websites as practice.
  • Modify the sites we made by the latest framework of the world.
  • Know about Domain, Hosting, IDE, GitHub etc.
